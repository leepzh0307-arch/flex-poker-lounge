# 项目铁则（AI必须100%遵守，不得修改）
## 1. 项目基础信息
- 项目名称：flex-poker 灵活扑克网页游戏
- 核心功能：德州扑克为主，房主可自定义发牌规则的网页联机扑克游戏
- 面向用户：纯小白，所有代码必须加中文注释，逻辑简单清晰
## 2. 技术栈铁则（不得更换）
- 前端：纯HTML5+CSS3+原生JavaScript（ES6+），不得用任何框架
- 后端：Node.js + Express + Socket.io（实现联机功能）
- 语音：声网Agora Web SDK
- 部署：前端Vercel，后端Render，代码托管GitHub

## 4. 视觉设计铁则（重点！必须遵守）
1. 所有颜色、字体、间距、圆角、按钮大小，**必须全部定义在themes/default.css的CSS变量里**
2. 其他CSS文件只能引用变量，绝对不能直接写颜色值、字体大小
3. 示例：必须用`background: var(--bg-primary);`，不能写`background: #1a2a1a;`
4. 页面布局必须响应式，电脑和手机都能正常显示
5. 扑克牌、按钮等元素必须有hover/点击的动画效果
## 5. 功能铁则
1. 必须只有2个页面：首页（输入昵称/房间号）、房间页（游戏游玩）
2. 核心角色：房主（创建房间的人，可自定义规则）、加入玩家
3. 必须实现的功能：
   - 房间创建/加入/退出
   - 实时联机，所有玩家的游戏状态实时同步
   - 德州扑克完整玩法（发牌、下注、弃牌、结算）
   - 房主专属面板：可自定义发牌顺序、发牌数量、公共牌/玩家牌分配
   - 在线实时语音通话，可开关麦克风/扬声器
4. 所有游戏逻辑必须前端+后端双重校验，防止作弊
5. 代码必须加详细中文注释，每个函数、每个变量都要说明是干嘛的
